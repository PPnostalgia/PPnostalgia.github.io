# 保守型

**保守型**：必须避免严重的错误或损失；**不必努力，不必烦恼，不必经常做决定，适合有一笔闲钱一直放那不用管的**。

**进取型**：愿意承担风险；愿意花许多时间小心地选择那些合理的且有吸引力的投资。

需要明确的一点是：投资者所追求的回报率依赖于投资者愿意和能够完成任务的明智的努力程度

# 投资

1. 如果投机，最终你将(可能)失去钱。
2. 当**大多数人**(包括专家)悲观时，买；而当他们相当乐观时，卖。
3. 调查，然后投资。

***投资和投机的区别***

投机是非理性的，具有以下特点：

（1）意识不到自己的投机行为；

（2）缺乏足够的知识和技巧，严重投机；

（3）加杠杆，动用比你能承受的损失更多的资金。

明确投资和投机的区别在于：投资看重定量分析，投机看重定性分析。投资看重对象将来能实现的利益，每一年每一季度的利润率，依据现在的数据情况，更有判断依据。投机就是觉得对象是一个很好的发展对象，只是猜测。

# 如何获益

## 整体策略

1. 购买基金代替自选股票，尽量购买被动型基金，如纳斯达克、标普500；可划出一小部分购买主动型基金来博取高额收益。
2. 定投：即美元成本平均值方法，其隐含着每月或每季度投资等额的资金于普通股上。用这种方法，在价格水平较低的股市上购买的股票就比水平较高时多，最终持有的所有股票就有满意的平均价格。
3. 方程式计划。核心是通过**固定规则**调整股债比例，以实现风险控制与收益平衡。本质是**用规则战胜人性**，通过机械化的资产再平衡，将市场波动转化为收益来源。
4. 动荡市场环境下（如2025年4月关税政策朝令夕改），可配置些黄金和货币基金代替债券

## 方程式计划

- **股票与债券的比例动态调整**：

  格雷厄姆建议防御型投资者将可投资资金在股票和债券之间进行分配，比例范围通常为**25%-75%**。具体比例根据市场估值灵活调整：

  - 当股市估值较高（如牛市）时，降低股票持仓比例至下限（如25%），增加债券比例；
  - 当股市估值较低（如熊市）时，提高股票持仓比例至上限（如75%），减少债券比例。

- **操作方式**：

  投资者可设定触发调整的阈值（例如股票每上涨20%则减持部分仓位），通过定期再平衡（如每年一次）维持目标比例，避免追涨杀跌。

------

意义：

- **对抗人性弱点**：

  通过规则化的操作减少主观判断，避免因市场情绪（如贪婪或恐惧）导致的非理性决策。例如，牛市中自动减仓可规避高位接盘风险，熊市中加仓则能捕捉低估机会10。

- **风险控制**：

  债券作为稳定器，在市场剧烈波动时提供缓冲；股票比例上限（如75%）则限制了极端下跌中的损失。

- **长期复利**：

  强制低买高卖的行为可能提升长期收益，尤其在市场周期性波动中增强收益的确定性。

# 止盈策略

基础：设置止盈目标收益率（10%-15%），达标后卖出50%仓位，剩余仓位使用“回撤止盈”：从最高点回撤5%时卖出剩余部分。

进阶：将策略分为**长期估值层**（核心）、**中期技术层**（趋势跟踪）、**短期波动层**（风险控制）

| **步骤** | **操作内容**                   | **频率**   |
| -------- | ------------------------------ | ---------- |
| 监控     | 每月检查基金净值和美股指数估值 | 每月1次    |
| 止盈     | PE>35倍或年化收益>20%时减持    | 触发条件时 |
| 止损     | 回撤>8%或持仓1年未盈利时卖出   | 触发条件时 |
| 再平衡   | 每年12月调整股债比例至7:3      | 每年1次    |

| **层级**   | **策略触发条件**            | **执行优先级** | **操作目标**    |
| ---------- | --------------------------- | -------------- | --------------- |
| 长期估值层 | PE突破历史阈值              | 最高           | 大比例止盈/清仓 |
| 中期技术层 | 跌破关键均线或趋势线        | 中等           | 部分止损/止盈   |
| 短期波动层 | 单日暴跌或波动率骤升（ATR） | 最低           | 紧急对冲或减仓  |

### **1. 目标收益率止盈法**

- **规则**：设定年化收益目标（如10%-15%），达标后分批止盈。
- 操作：
  - 首次达标时卖出50%，剩余仓位设定回撤止盈（如从高点回撤5%时清仓）。
  - 示例：若基金年化收益达20%，先卖出50%，剩余部分若回撤5%则全部卖出。

### **2. 估值区间止盈法**

- **规则**：参考美股指数历史PE分位数（如纳斯达克PE>35倍时止盈）。
- 操作：
  - 通过Wind App查询指数估值，当PE超过历史70%分位时减持30%，超过80%分位时再减持30%。
  - **替代方案**：若无法获取实时PE数据，可观察市场情绪（如媒体频繁报道美股创新高时逐步减仓4）。

### **3. 定期再平衡止盈**

- **规则**：每年或每半年调整股债比例（如70%美股指数基金+30%债券基金）。
- 操作：
  - 强制卖出超配的美股基金，买入债券基金锁定收益，降低风险

# 止损策略

### **1. 最大回撤止损法**

- **规则**：单笔投资最大亏损容忍度设为总本金的5%-8%。
- 操作：
  - 若基金净值从买入价下跌8%，则一次性止损离场。
  - **注意**：场外基金净值延迟，需以T+1日确认的净值为准。

### **2. 时间止损法**

- **规则**：持仓超过设定时间（如1年）未达预期收益则止损。
- 操作：
  - 若持有1年后仍亏损，卖出并转换至低风险资产（如货币基金）。

### **3. 技术信号辅助止损**

- **规则**：通过简单技术指标（如200日均线）判断趋势。
- 操作：
  - 若纳斯达克指数跌破200日均线且维持3个交易日，视为趋势破位，卖出场外基金。
  - **替代方案**：关注财经新闻中“美股技术性熊市”报道，作为止损信号

# 《巴菲特致股东的信》的建议

一句话总结：以合理甚至低估的价格，购买那些你能够理解的、具有持久竞争优势的、由优秀管理层经营的优质企业的股权，并打算长期持有，同时避免使用杠杆，若无暇研究个股，则定投低成本被动型指数基金。

1. 坚守“能力圈” (Circle of Competence)：
   - **核心思想：** 只投资于自己能够理解的行业和公司。保守型投资者不应涉足那些业务模式复杂、难以预测或自己不熟悉的领域。
   - **实践：** 如果你无法用简单的语言向一个10岁的孩子解释清楚一家公司的业务和盈利模式，那么这家公司可能就在你的能力圈之外。
2. 强调“安全边际” (Margin of Safety)：
   - **核心思想：** 以远低于其内在价值的价格购买资产。这是风险管理的核心，为可能发生的错误判断、坏运气或宏观经济冲击提供缓冲。
   - **实践：** 保守型投资者在估值时应更为谨慎，要求更大的折扣。例如，如果一家公司内在价值是100元，保守投资者可能希望在60-70元甚至更低时才买入。
3. 长期持有优质企业 (Long-Term Holding of Quality Businesses)：
   - **核心思想：** 将股票投资视为购买企业的一部分所有权，而非短期交易的筹码。选择那些具有持续竞争优势（“护城河”）、良好管理层和稳健财务状况的优质企业，并长期持有。
   - **实践：** 不要试图预测市场短期波动，专注于企业的长期盈利能力和价值增长。频繁交易会增加成本并可能导致情绪化决策。
4. 视市场波动为机遇而非风险 (View Market Volatility as Opportunity, Not Risk)：
   - **核心思想：** 巴菲特认为真正的风险是永久性资本损失的风险，而非暂时的股价下跌。市场先生情绪化时，会提供以低价买入优质资产的机会。
   - **实践：** 保守型投资者应利用市场恐慌时出现的低估机会增持优质股票，而不是在恐慌中抛售。
5. 避免使用杠杆 (Avoid Leverage)：
   - **核心思想：** 杠杆会放大收益，但同样会放大亏损。对于保守型投资者而言，借钱投资会显著增加风险，可能导致在不利情况下被迫清算。
   - **实践：** 用自有资金进行投资，不要进行保证金交易或借贷投资。
6. 对于大多数人，投资低成本指数基金 (For Most People, Invest in Low-Cost Index Funds)：
   - **核心思想：** 巴菲特多次强调，对于那些没有时间、专业知识或兴趣去挑选个股的投资者（包括许多保守型投资者），定期投资于低成本的S&P 500指数基金是最佳选择。
   - **实践：** 这种策略提供了广泛的分散化，避免了个股选择的风险，并能以极低的成本分享美国（或其他市场）整体经济的增长。
7. 注重企业管理层的品质 (Focus on the Quality of Management)：
   - **核心思想：** 投资于由诚实、有能力且为股东着想的管理层领导的公司。好的管理层能有效地配置资本，并在困难时期带领企业渡过难关。
   - **实践：** 在分析公司时，考察管理层的过往业绩、资本配置决策以及对待股东的态度。
8. 保持简单，避免复杂 (Keep it Simple, Avoid Complexity)：
   - **核心思想：** 复杂的投资策略和金融产品往往隐藏着未知的风险，并且难以理解。
   - **实践：** 保守型投资者应坚持简单、透明的投资方式，避免那些自己无法完全理解的衍生品、结构化产品等。
9. 现金的价值 (The Value of Cash)：
   - **核心思想：** 在缺乏有吸引力的投资机会时，持有现金并非坏事。现金提供了在市场低迷时抓住机会的“弹药”。
   - **实践：** 不要因为害怕“踏空”而强迫自己投资。保守型投资者应有耐心等待合适的时机。
10. 持续学习，独立思考 (Continuous Learning and Independent Thinking)：
    - **核心思想：** 不要盲从市场噪音、专家意见或短期趋势。通过阅读和思考，形成自己的投资判断。
    - **实践：** 阅读公司年报、行业报告，学习会计知识，理解商业模式。
